Zwracane wartości

LibraryBook

Książka w postaci pliku JSON' wygląda następująco

{

"id": 960,

"title": "Angels & Demons (Robert Langdon #1)",

"authors": "Dan Brown",

"rating": 3.89,

"popularity": 93.02830769230769,

"pages": 736,

"year": "2006-04-01",

"publisher": "Pocket Books"

}

LibraryUser

Użytkownik w postaci pliku JSON' wygląda następująco

{

"id": "id_użytkownika_w_formie_UUID",

"username": "nazwa_użytkownika",

"email": "uzytkownik@mail.com"

}

UserHistory

Historia wypożyczeń w postaci pliku JSON' wygląda tak jak poniżej. Jeżeli pole "returnedDate" jest równe null oznacza, że książka nie została jeszcze zwrócona.

{

"userId" : "id_użytkownika_w_formie_UUID",

"title": "Harry Potter and the Half-Blood Prince (Harry Potter #6)",

"bookCopyId": 23,

"borrowDate": "2020-12-21",

"returnedDate": null

}
